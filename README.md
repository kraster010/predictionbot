# Prediction Bot

Track Predictions on BitcoinMarkets

## Getting Started

### Prerequisites

Install

```
Python 2.x
pip
```

### Installing

```
pip install -r requirements.txt
```

inside your project folder:

```
http://127.0.0.1:8000/
```

Open browser and go to:

- [localhost](http://127.0.0.1:8000/):

```
http://127.0.0.1:8000/
```
