# Prediction Bot

Track Predictions on BitcoinMarkets

## Getting Started

### Prerequisites


```
Python 2.x
pip
```

### Installing

```
pip install -r requirements.txt
```

### Running

inside your project folder:

```
python manage.py migrate
python manage.py runserver
```

Open browser and go to:

- [localhost](http://127.0.0.1:8000/):

```
http://127.0.0.1:8000/
```
